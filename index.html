O mně

Jsem začínající vývojář softwaru s vášní pro IT a programování. Od prvotního zájmu o počítačové hry mě cesta dovedla k vývoji softwaru, kde mohu přetvářet nápady v reálné aplikace. Rozšiřuji své znalosti v oblasti programování webových a desktopových aplikací, databází, automatizovaného testování a DevOps nástrojů. Nejvíce se zaměřuji na C#, Javu a webové technologie. Mým cílem je růst jako vývojář a přispívat k inovativním projektům, které mají smysl.

Projekty

Seznamka Bonded

Bootstrap, CSS, Typescript, ASP.NET, MS/SQL, Entity Framework, Node-JS

Bonded je seznamka založená na osobnostním testu, který pokrývá 13 psychologických kategorií. Na základě bodového skóre algoritmus vyhodnocuje kompatibilitu a doporučuje nejvhodnější partnery.

Databáze uživatelů

C# ASP.NET, MS/SQL, Entity Framework

Tento projekt se zaměřuje na správu uživatelských dat, s důrazem na efektivní ukládání a získávání informací. Implementoval jsem dynamické prvky, formulářovou validaci a interaktivní výpočty v reálném čase.

Akvárium

Java, Spring

V tomto projektu jsem si osvojil práci s Java a Spring frameworkem. Vyzkoušel jsem si práci s komponentami, stavem aplikace a základními principy moderního backendu.

Počasí

Java, Spring

V tomto projektu jsem se zaměřil na získávání a zobrazování dat o počasí pomocí Java a Spring. Implementoval jsem integraci s externími API a zpracování dat.

Digitální výstava

Bootstrap, SCSS

Při tvorbě této fiktivní stránky jsem pracoval s použitím frameworku Bootstrap. Zaměřil jsem se na responzivní design, grid systém a komponenty pro rychlé prototypování moderního webu.

Formulář

Bootstrap, Javascript

Pomocí JavaScriptu jsem v tomto projektu implementoval dynamické prvky, formulářovou validaci a interaktivní výpočty v reálném čase.

Akvárium

React, Bootstrap, JavaScript

Zde jsem pomocí Reactu pracoval s komponentami. Vyzkoušel jsem si práci se stavem aplikace, props a základní principy moderního frontendu.

Databáze knih

PHP, MYSQL, Bootstrap

V tomto projektu jsem si osvojil práci s PHP a MySQL. Vytvořil jsem systém pro správu knihovny, kde se pracuje s databází – přidávání, mazání a úprava knih. Naučil jsem se základy CRUD operací, práci s formuláři a komunikaci mezi frontendem a databází.
